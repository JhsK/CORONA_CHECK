# CORONA-CHECK 

- 타입스크립트 개념 공부 후 응용 목적인 간단한 웹사이트
- 보건복지부_코로나19감염현황 조회 서비스 API를 이용하여 데이터를 표현
- Chart.js, typescript, eslint 숙련도 향상을 목표

<br>

## 시연 방법

1. back 폴더 터미널 접근
2. npm run dev 입력 ( 코로나 API 요청 후 받아오는 API 서버 )
3. index.html 파일 live server로 실행 

<br>
